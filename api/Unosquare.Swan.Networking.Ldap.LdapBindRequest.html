    Type Name Description
    System.Int32 version

    The Ldap protocol version, use Ldap_V3. Ldap_V2 is not supported.
    System.String dn

    If non-null and non-empty, specifies that the connection and all operations through it should be authenticated with dn as the distinguished name.
    System.SByte[] passwd

    If non-null and non-empty, specifies that the connection and all operations through it should be authenticated with dn as the distinguished name and passwd as password.
    LdapControl[] cont

    Any controls that apply to the simple bind request, or null if none.
